# Magento2-PT-BR
Tradução Magento2 Português Brasil

## Como Instalar

### Via Composer

```sh
composer require bleez/magento2-pt_br
```

### Manual 

- Baixe o Zip
- Crie as pastas i18n/bleez/ no diretório app
- Extraia os arquivos

## Como Contribuir

- Não use Google Translate
- Fork o repositorio e envie um pull request ou edite no github
